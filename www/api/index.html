AX2JS API

Tree

Get all elements: /api/tree

Get all elements of a specific type: /api/tree/class

Get all methods of a class: /api/tree/class/SPBusinessLogic


Element

Get element by id: /api/element/3

Get element source code (classes and tables) by id: /api/element/4/source

Get single method element source code (classes and tables) by id: /api/element/4/source/fillShows

Get element source X++ (classes and tables) by id: /api/element/4/source/xpp

Get element source AST (classes and tables) by id: /api/element/4/source/ast

Compile element: /api/element/4/compile

Get properties of element: /api/element/2/props

Get list and source code of all dependencies of an element (including for the element itself): /api/element/4/depsource


All element ids can be replaced by "type/name" like: /api/element/class/SPBusinessLogic. It is not case sensitive.
